# note-mark application

## TechStack: Electron, React, Tailwindcss, TypeScript

## Create project

```
yarn create @quick-start/electron
react -> typescript -> no -> no
yarn
# setup tailwindcss
yarn add -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
